# SMS Spam Detection Web App 🚀

A Django-based web application that detects SMS spam in real-time using Machine Learning.

## Features
- Real-time spam/ham classification
- Random Forest model with 97% accuracy
- TF-IDF vectorization for text processing
- Clean web interface built with Django

## Tech Stack
Python | Django | Scikit-learn | Machine Learning | HTML/CSS

## How it works
1. User enters an SMS message
2. Model preprocesses the text using TF-IDF vectorizer
3. Random Forest classifier predicts spam or ham
4. Result is displayed instantly
